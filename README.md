# App de Streaming de Áudio (Java)

Projeto desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos.

Simula um sistema simples de streaming de áudio, com músicas e podcasts, permitindo reproduções, curtidas e classificação dos conteúdos com base em engajamento.

## Funcionalidades

- Reprodução de músicas e podcasts  
- Registro de curtidas  
- Classificação de áudios  
- Identificação de conteúdos populares  

## Conceitos aplicados

- Encapsulamento  
- Herança  
- Polimorfismo  
- Sobrescrita de métodos  

## Tecnologias

- Java  

## Como executar

1. Clone o repositório  
2. Abra em uma IDE (IntelliJ ou Eclipse)  
3. Execute a classe `Principal`
